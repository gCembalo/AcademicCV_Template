## Description

This is a basic academic CV written during my MSc in theoretical physics seen the necessity to apply to some Summer School. This template, as you can easly verify, it's inspired by Dubasi Pavan Kumar's template, that you can find here on Overleaf, simply searching his name.


## How to use

To complete your CV you can simply compile all the fields present in "Data.sty" file. There you'll find all the different CV's section. If you want to personalize the specific commands you can see the "Packages.sty" file, where all the environment (formatting option included) are well defined. Here, in the "main.tex" you can only find the imported packages, data file and the launching of the commands; It's in this file that you can remove/add some section simply commenting/uncommenting the command.

Note. It's possible that you have to adjust some part of the code in "Packages.sty" file, because you can need different number / different specifics of command.
